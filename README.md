# 🥗 Gestational Diabetes Meal Planner

An educational Streamlit application designed to help pregnant women with gestational diabetes receive nutrition guidance and meal suggestions.

## Features

- BMI Calculator
- Daily Calorie Estimate
- Blood Glucose Assessment
- Pregnancy-Friendly Meal Suggestions
- Healthy Snack Recommendations
- Foods to Limit Guidance

## Technologies Used

- Python
- Streamlit

## Disclaimer

This application is for educational purposes only and does not provide medical advice. Users should consult their healthcare provider for diagnosis and treatment decisions.
